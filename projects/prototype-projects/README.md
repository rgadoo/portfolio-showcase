## Overview

This repository showcases a variety of prototype projects built with different technologies and for different purposes. Each project demonstrates specific technical skills and problem-solving approaches.

## Projects

### [Nandi Spiritual Platform](./nandi-platform/)

A multi-module spiritual platform built with Domain-Driven Design principles, featuring three interactive modules for Vedic wisdom exploration.

**Key Highlights:**
- Domain-Driven Design architecture
- Flask backend + React frontend
- Three distinct modules (KarmaCafe, SoulQuest, WisdomPets)
- OpenAI integration for AI-powered conversations
- Modular, maintainable codebase

**Status:** Prototype

### [Personal Activity Tracker](./personal-tracker/)

An NLP-powered web application for tracking and analyzing personal activities with automatic categorization and sentiment analysis.

**Key Highlights:**
- NLP processing with NLTK and spaCy
- Automatic activity categorization
- Sentiment analysis
- Multi-user support
- Privacy-focused design

**Status:** Prototype

### [Vedic Astrology Calculator](./vedic-astrology/)

A birth chart calculator that computes and visualizes Vedic (sidereal) astrological charts using high-precision Swiss Ephemeris calculations.

**Key Highlights:**
- Swiss Ephemeris integration
- High-precision planetary calculations
- Interactive birth chart visualization
- Vedic astrology calculations (Lahiri Ayanamsa)
- Timezone support

**Status:** Prototype

### [Stock Analysis App](./stock-analysis/)

An AI-powered stock market analysis application combining technical analysis with machine learning for price prediction.

**Key Highlights:**
- LSTM neural network for price prediction
- Gann analysis for technical indicators
- Multi-stack architecture (Python FastAPI + Java)
- Interactive charting (Matplotlib, Plotly)
- Historical data analysis

**Status:** Prototype

### [TradingView Integration](./trading-view/)

Tools and utilities for integrating with and extending TradingView functionality.

**Key Highlights:**
- TradingView API integration
- Data processing and analysis
- Strength data analysis

**Status:** Prototype
